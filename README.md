# iwacai_lottery_app

本应用基于flutter开发，使用getx进行状态管理，包含丰富的赛事，五大联赛、世界杯等众多世界主流赛事；后端使用springboot mybatis redis mysql进行开发。

本app有许多特色功能，欢迎star、fork学习交流，如果体育赛事应用开发有需求请联系(QQ:625751973 或者 Telegram @jpchou 一起交流探讨)
